Features

    Spyral is an application that should allow you to take images and videos of space in real time and view a panorama version of it.

    Spyral allows you to capture the images and videos by fetching the data from a NASA API storage and creating one big 360 image (panorama) version of it by stitching them using algorithm that recognizes patterns in the images captured.

    Images displayed are the latest ones,most recently captured by NASA satelites.
